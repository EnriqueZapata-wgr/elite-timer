## 📝 Checklist post-generación

Para cada imagen:

1. Iterar 4-6 versiones en MJ con el prompt
2. Elegir la mejor (criterio: composición editorial, contraste B/N alto, sin elementos clichés)
3. Descargar PNG en alta resolución
4. Renombrar con el filename exacto del documento (lowercase, guiones)
5. Mover a la carpeta correspondiente:
   - `assets/images/habits-portal/` (crear si no existe)
   - `assets/images/health-hub/` (crear si no existe)
   - `assets/images/electrons/` (ya existe)
6. Correr `npm run optimize-images` para comprimir
7. `git add assets/images/ && git commit -m "feat(assets): imágenes B/N hábitos + historia + electrones nuevos"`
8. `git push origin main` (o branch de imágenes si quieres separar)

Después: mini-sprint a CC para cablear los `require()` en habits-portal.tsx, health-hub.tsx, y HoyEditorialSection.tsx + crear los 3 electrones nuevos en `electrons.ts`.
