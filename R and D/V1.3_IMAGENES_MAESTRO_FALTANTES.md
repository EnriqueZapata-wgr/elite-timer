## 📋 Checklist post-generación

Para CADA imagen:

1. [ ] Iterar 4-6 versiones en MJ con el prompt exacto
2. [ ] Elegir la winner (composición editorial premium, contraste B/N alto, sin clichés/branded)
3. [ ] Descargar PNG alta resolución (MJ entrega típicamente 2048×1146)
4. [ ] Renombrar EXACTAMENTE como el filename del doc (lowercase, guiones, .png)
5. [ ] Mover a la carpeta correspondiente

Después de todas:

6. [ ] **Copias:**
```powershell
cp assets/images/hoy-extra/ayuno.png assets/images/habits-portal/ayuno.png
cp assets/images/hoy-extra/sueno.png assets/images/habits-portal/sueno.png
```

7. [ ] **Comprimir batch:** `npm run optimize-images`
8. [ ] **Verificar tamaños:** cada PNG <500KB después de optimize-images
9. [ ] **Commit:**
```powershell
git add assets/images/
git commit -m "feat(assets): 18 imágenes B/N editoriales — hábitos pilares + historia clínica + electrones nuevos + ciclo + embarazo"
git push origin main
```

10. [ ] **Avisarme** para escribir el mini-sprint a CC que cablea TODOS los `require()` en:
