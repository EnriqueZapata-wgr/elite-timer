## 1. Contexto y objetivo

Las EditorialCards del HOY ya quedaron visualmente perrísimas (imágenes B/N + gradient diagonal + aspectRatio 16:9 — sprint anterior). Pero el tab HOY está **revuelto funcionalmente**:

- Las cards editoriales son **decorativas** — solo enlazan, no completan electrones desde la card
- Conviven con el sistema viejo (PRÓXIMO ELECTRÓN con botón COMPLETAR, grid de electrones, secciones sueltas de Proteína/Agua/Ayuno, Agenda triple Mañana/Tarde/Noche)
- Faltan elementos críticos: barra visual en Proteína/Agua, botones +250/+500 en Agua, card de Ayuno con timer LIVE, contador de electrones visible por card

Este sprint consolida HOY como **feed editorial funcional**: las cards SON el sistema, no su sombra.

### 4.4 HoyEditorialSection: completar boolean desde card

Archivo: `src/components/hoy/HoyEditorialSection.tsx`

Para cada card booleana (luz_solar, meditacion, bano_frio, grounding, lentes_rojos, checkin), cambiar el `onTap` de "enlazar a ruta" a "toggle electron":

```typescript
const toggleBoolean = async (cardKey: string) => {
  if (!userId) return;
  const source = CARD_TO_ELECTRON[cardKey];
  const isCompleted = boolBySource.get(source)?.completed;
  if (isCompleted) {
    await revokeBooleanElectron(userId, source);
  } else {
    await awardBooleanElectron(userId, source);
  }
  DeviceEventEmitter.emit('electrons_changed');
};
```

Excepciones (mantener enlace, NO toggle):
- `meditacion` → `/meditation` (porque requiere sesión real verificada)
- `fuerza` → `/log-exercise` (verified)
- `breathwork` → `/breathing` (verified)
- `suplementos` → la tabla abajo lleva el detalle, card solo enlaza a `/supplements`
