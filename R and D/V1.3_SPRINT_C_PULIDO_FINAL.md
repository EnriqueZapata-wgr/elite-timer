### 2.1 EditorialCard: círculo checkable en vez de emoji

Archivo: `src/components/hoy/EditorialCard.tsx`

**Cambios:**
- Prop `icon` queda OPCIONAL pero ya no se renderiza como emoji en el topRow
- Agregar componente interno `<CheckCircle done={state === 'done'} />` que renderiza:
  - **Pending:** círculo vacío 28×28, borde 2px `rgba(255,255,255,0.8)`, fondo transparente
  - **Done:** círculo lleno 28×28, fondo `#A8E02A` (lima), checkmark blanco centrado (SVG o emoji ✓ con fontSize 16, color #000)
- Posición: misma esquina sup-izq donde estaba el emoji, dentro del topRow
- El emoji se elimina del render — los specs en `hoy-cards.ts` mantienen su `icon` por compat (sin uso visual ya)

**Implementación del círculo:**
```tsx
function CheckCircle({ done }: { done: boolean }) {
  return (
    <View style={[
      styles.checkCircle,
      done && styles.checkCircleDone,
    ]}>
      {done ? <EliteText style={styles.checkMark}>✓</EliteText> : null}
    </View>
  );
}

// styles:
checkCircle: {
  width: 28, height: 28, borderRadius: 14,
  borderWidth: 2, borderColor: 'rgba(255,255,255,0.8)',
  backgroundColor: 'transparent',
  alignItems: 'center', justifyContent: 'center',
},
checkCircleDone: {
  backgroundColor: ATP_BRAND.lime,
  borderColor: ATP_BRAND.lime,
},
checkMark: { color: '#000', fontFamily: Fonts.bold, fontSize: 16, lineHeight: 18 },
```

**Comportamiento del círculo:**
- Cards booleanas (toggle desde card): círculo refleja el state real, tap en la card completa/revoca (igual que ahora)
- Cards de enlace (MEDITACIÓN/FUERZA/BREATHWORK/SUPLEMENTOS): círculo refleja si su electrón asociado está done para el día
- Cards cuantitativas (PROTEÍNA/AGUA): círculo done cuando `current >= target`
- Card AYUNO: círculo done cuando hay ayuno completado hoy
- Cards sin electrón asociado (UV, CARDIO, PASOS): NO renderizar círculo (son informativas)

Agregar prop opcional `showCheckCircle?: boolean` (default true cuando hay state, false cuando no). Pasarlo `false` desde HoyEditorialSection en UV/CARDIO/PASOS y desde YoEditorialSection en CRONOTIPO/RANK/TESTS/REPORTES/etc (todas las cards de YO son informativas, no completables).
